# Coolie Powerhouse Fish Game

## How to Add Your Coolie Powerhouse Song

1. **Download your coolie powerhouse song** (MP3 or OGG format)
2. **Place the file** in the same folder as `fish.html`:
   - Save as `coolie-powerhouse.mp3` or `coolie-powerhouse.ogg`
   - Or rename your file to match these names
3. **Refresh the game** - the song will automatically play as background music
